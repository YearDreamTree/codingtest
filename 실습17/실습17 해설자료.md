## β… `make_team` ν•¨μ κµ¬ν„ μƒμ„Έ (μ •λ ¬ + ν¬μΈν„° λ°©μ‹)

μ •λ‹µ μ½”λ“(λ‘ λ²μ§Έ)κ°€ μ„ μ•„μ΄λ””μ–΄λ¥Ό μ–΄λ–»κ² κµ¬ν„ν•λ”μ§€ μƒμ„Έν λ³΄κ² μµλ‹λ‹¤.

1.  **μ΄κΈ° μ„¤μ • λ° μ •λ ¬**:

    ```python
    selected = [False] * n # λ¨λ“  μ„ μλ¥Ό 'μ„ νƒ μ• λ¨'μΌλ΅ μ΄κΈ°ν™”

    # κ° μ”μ†λ¥Ό (μ›λ³ΈμΈλ±μ¤, μ΄λ¦„, μν…) νν”λ΅ λ§λ“¦
    # key=lambda x: -x[2] : νν”μ 3λ²μ§Έ μ”μ†(x[2]=μν…)μ μμκ°’ κΈ°μ¤€ μ •λ ¬ -> μν… λ‚΄λ¦Όμ°¨μ
    sorted_a = sorted([(i, p[0], p[1]) for i, p in enumerate(players)], key=lambda x: -x[2])
    # μ„μ™€ λ™μΌν•κ² λ“λ¦¬λΈ” λ¥λ ¥μΉ κΈ°μ¤€ λ‚΄λ¦Όμ°¨μ μ •λ ¬
    sorted_b = sorted([(i, p[0], p[2]) for i, p in enumerate(players)], key=lambda x: -x[2])

    result_a = [] # μ—λ¦¬μ¤ ν€ κ²°κ³Ό λ¦¬μ¤νΈ
    result_b = [] # μ²΄μ…” ν€ κ²°κ³Ό λ¦¬μ¤νΈ

    idx_a = 0 # sorted_a λ¦¬μ¤νΈλ¥Ό κ°€λ¦¬ν‚¬ ν¬μΈν„°
    idx_b = 0 # sorted_b λ¦¬μ¤νΈλ¥Ό κ°€λ¦¬ν‚¬ ν¬μΈν„°
    ```
    * `enumerate(players)`λ¥Ό ν†µν•΄ κ° μ„ μμ μ›λ μΈλ±μ¤ `i`μ™€ μ •λ³΄ `p`λ¥Ό μ–»μµλ‹λ‹¤.
    * λ¦¬μ¤νΈ μ»΄ν”„λ¦¬ν—¨μ…μΌλ΅ `(i, p[0], p[1])` ν•νƒμ νν” λ¦¬μ¤νΈλ¥Ό λ§λ“­λ‹λ‹¤.
    * `sorted()` ν•¨μμ™€ `key=lambda x: -x[2]`λ¥Ό μ΄μ©ν•μ—¬ λ¥λ ¥μΉ κΈ°μ¤€ λ‚΄λ¦Όμ°¨μ μ •λ ¬λ λ¦¬μ¤νΈ `sorted_a`, `sorted_b`λ¥Ό μƒμ„±ν•©λ‹λ‹¤.

2.  **μ„ μ μ„ λ° λ£¨ν”„**:

    ```python
    for _ in range(n // 2):

        # sorted_a[idx_a] κ°€ κ°€λ¦¬ν‚¤λ” μ„ μκ°€ μ΄λ―Έ λ½‘ν”λ”μ§€(selected) ν™•μΈ
        # λ½‘ν”λ‹¤λ©΄(True), λ‹¤μ ν›„λ³΄λ¥Ό λ³΄κΈ° μ„ν•΄ idx_a μ¦κ°€ (while λ°λ³µ)
        while selected[sorted_a[idx_a][0]]: # sorted_a[idx_a][0]μ€ ν•΄λ‹Ή μ„ μμ 'μ›λ³Έ μΈλ±μ¤'
            idx_a += 1
        # whileμ„ λΉ μ Έλ‚μ™”λ‹¤λ©΄, λ½‘νμ§€ μ•μ€ μµκ³  μν… λ¥λ ¥ μ„ μλ¥Ό μ°Ύμ€ κ²ƒ!
        best_a_player_index = sorted_a[idx_a][0] # λ½‘μ„ μ„ μμ μ›λ³Έ μΈλ±μ¤
        best_a_player_name = sorted_a[idx_a][1]  # λ½‘μ„ μ„ μμ μ΄λ¦„
    ```

    ```python
    return result_a, result_b # λ‘ λ¦¬μ¤νΈλ¥Ό νν”λ΅ λ¬¶μ–΄ λ°ν™
    ```

## β… (μ°Έκ³ ) λ‹¤λ¥Έ μ ‘κ·Ό λ°©μ‹: ν™ (Heapq)

ν™(Heap) μλ£κµ¬μ΅°, νΉν νμ΄μ¬μ `heapq` λ¨λ“(μµμ† ν™ μ κ³µ)μ„ μ΄μ©ν•  μλ„ μμµλ‹λ‹¤.

1.  **ν™ μƒμ„±**: μν… λ¥λ ¥μΉ κΈ°μ¤€ μµλ€ ν™(`a`)κ³Ό λ“λ¦¬λΈ” λ¥λ ¥μΉ κΈ°μ¤€ μµλ€ ν™(`b`)μ„ λ§λ“­λ‹λ‹¤. νμ΄μ¬ `heapq`λ” μµμ† ν™μ΄λ―€λ΅, μµλ€ ν™ ν¨κ³Όλ¥Ό λ‚΄κΈ° μ„ν•΄ **λ¥λ ¥μΉμ— μμ λ¶€νΈ(-)** λ¥Ό λ¶™μ—¬μ„ `(-λ¥λ ¥μΉ, μ΄λ¦„, μ›λ³ΈμΈλ±μ¤)` ν•νƒλ΅ ν™μ— `heappush` ν•©λ‹λ‹¤. μ΄λ ‡κ² ν•λ©΄ ν™μ—μ„ κ°’μ„ κΊΌλ‚Ό λ•(`heappop`) μμκ°’μ΄ κ°€μ¥ μ‘μ€ κ²ƒ(μ›λλ” μ–‘μκ°’μ΄ κ°€μ¥ ν° κ²ƒ)μ΄ λ¨Όμ € λ‚μ¤κ² λ©λ‹λ‹¤.
2.  **μ„ μ μ„ λ°**: κ° ν„΄λ§λ‹¤ ν•΄λ‹Ή ν™(μ—λ¦¬μ¤λ” `a`, μ²΄μ…”λ” `b`)μ—μ„ `heappop`μΌλ΅ μµκ³  λ¥λ ¥μΉ ν›„λ³΄λ¥Ό κΊΌλƒ…λ‹λ‹¤. κΊΌλ‚Έ ν›„λ³΄μ μ›λ³Έ μΈλ±μ¤λ΅ `selected` λ°°μ—΄μ„ ν™•μΈν•μ—¬ μ΄λ―Έ λ½‘ν μ„ μλ©΄, λ½‘νμ§€ μ•μ€ μ„ μκ°€ λ‚μ¬ λ•κΉμ§€ λ‹¤μ‹ `heappop`μ„ λ°λ³µν•©λ‹λ‹¤. λ½‘νμ§€ μ•μ€ μ„ μκ°€ λ‚μ¤λ©΄ `selected`λ¥Ό `True`λ΅ λ°”κΎΈκ³  ν€ λ¦¬μ¤νΈμ— μ΄λ¦„μ„ μ¶”κ°€ν•©λ‹λ‹¤.

ν™μ„ μ΄μ©ν•λ” λ°©μ‹λ„ μ‹κ°„ λ³µμ΅λ„λ” O(N log N)μΌλ΅ μ •λ ¬ λ°©μ‹κ³Ό λ™μΌν•λ©°, λ©”λ¨λ¦¬ μ‚¬μ©λ‰ λ“±μ—μ„ μ•½κ°„μ μ°¨μ΄κ°€ μμ„ μ μμµλ‹λ‹¤.

## β… μ „μ²΄ μ½”λ“ κµ¬ν„ (μ •λ ¬ + ν¬μΈν„° λ°©μ‹)

```python

    # κ° μ„ μκ°€ μ„ νƒλμ—λ”μ§€ μ—¬λ¶€λ¥Ό μ €μ¥ν•  λ¦¬μ¤νΈ (μ›λ³Έ μΈλ±μ¤ κΈ°μ¤€)
    selected = [False] * n

    # μ„ μ λ¦¬μ¤νΈλ¥Ό μν… λ¥λ ¥μΉ κΈ°μ¤€ λ‚΄λ¦Όμ°¨μμΌλ΅ μ •λ ¬
    # κ° μ”μ†λ” (μ›λ³Έ μΈλ±μ¤, μ΄λ¦„, μν… λ¥λ ¥μΉ) νν”
    sorted_a = sorted([(i, p[0], p[1]) for i, p in enumerate(players)], key=lambda x: -x[2])

    result_b = [] # μ²΄μ…” ν€ λ…λ‹¨

    return (result_a, result_b)
```
